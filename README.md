# ConNetwork

1. Install Android Studio, Nodejs & Java SDK 8 or above.
2. Run
```shell
$ npm install
$ npm install -g @ionic/cli
```
4. cd to ConNetwork folder
5. Run
```shell
$ ionic serve
```
